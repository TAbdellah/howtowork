📚 Guide de téléchargement

Accéder aux Cours depuis GitHub en Mode Offline

👥 Ce guide est destiné à :

Enseignants sans connexion internet dans leur salle informatique
Enseignants disposant d'une connexion internet dans leur salle informatique
Tous les utilisateurs souhaitant télécharger et consulter les ressources pédagogiques en local

Introduction

🎯 Objectif

Ce guide vous permet de télécharger facilement le référentiel de cours depuis GitHub et de l'utiliser localement sur votre ordinateur, qu'avec ou sans connexion internet.

Étapes d'Installation

1

Accéder au Référentiel

Visitez le lien du cours sur GitHub :

https://github.com/TAbdellah/cours
2

Télécharger le fichier ZIP

Cliquez sur le bouton vert « Code » en haut à droite, puis sélectionnez « Download ZIP ». Le téléchargement démarre automatiquement. Trouvez le fichier ZIP téléchargé dans votre dossier « Téléchargements ».

3

Extraire le contenu

Cliquez droit sur le fichier ZIP et sélectionnez « Extraire tout... » (Windows) ou « Décompresser » (Mac).

4

Ouvrir les fichiers

Naviguez dans le dossier extrait et ouvrez les fichiers HTML dans votre navigateur web. Les contenus sont maintenant accessibles hors ligne !

💡 Conseils Pratiques

Pour une meilleure expérience :

✓ Conservez le dossier extrait dans un endroit facilement accessible
✓ Créez un raccourci vers le fichier index.html principal
✓ Utilisez un navigateur moderne (Chrome, Firefox, Edge)
✓ Mettez régulièrement à jour vos ressources en téléchargeant la dernière version
